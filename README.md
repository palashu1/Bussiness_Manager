# Bill Tracker

## Overview

Bill Tracker is a web-based business management application designed to help small businesses manage customers, products, purchases, and sales invoices efficiently.

The application simplifies daily business operations by providing a centralized platform to track inventory, manage customer information, and generate sales invoices.

## Live Demo

https://bussinessmanager20260619150607.azurewebsites.net/

---

## Features

### Customer Management

* Add new customers
* Edit customer details
* Delete customers
* View customer list

### Product Management

* Add products
* Update product information
* Manage product pricing
* Track inventory quantity

### Sales Management

* Create sales invoices
* Select multiple products in a single invoice
* Calculate invoice totals automatically
* Support multiple payment methods

### Dashboard

* Quick access to core modules

## Getting Started

### Prerequisites

* Visual Studio 2022
* SQL Server
* .NET 8 SDK

### Installation

1. Clone the repository
2. Open the solution in Visual Studio
3. Configure the database connection string
4. Run database migrations
5. Start the application
